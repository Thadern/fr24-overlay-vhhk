Hong Kong FIR (VHHK) FR24 Overlay

This tool adds Hong Kong air traffic routes on top of FlightRadar24 map. Desktop version only.

DO NOT USE FOR FLIGHT PLANNING OR NAVIGATION

Information may contain errors, and may not be updated. Routes have been simplified for easier reading.

Installation 使用方法

  1. Drag this link into your browser's bookmark bar. (or right click then "Bookmark this link")

    將 此連結 拉到你的瀏覽器的書籤工具列上。 (或按右鍵選擇加到書籤)

  2. Open http://www.flightradar24.com

    打開 http://www.flightradar24.com

  3. Click on the bookmark

    按一下該書籤

  4. Set what to show on the left panel.

    在左面界面設定顯示模式。

Features

  1. Hong Kong SIDs, STARs and approach (07 / 07 Night / 25)
  2. Hong Kong Terminal Transition Routes
  3. Macau SIDs, STARs and approach
  4. Macau / Shenzhen / Guangzhou TTRs (J101, J103, J104)
  5. Holding Patterns
  6. FIR boundary
  7. Nearby routes
  8. ATC Radio
